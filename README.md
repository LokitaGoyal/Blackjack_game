# Blackjack Game

A simple Blackjack game built with Python.

## Features

- Player vs Dealer
- Random card distribution
- Hit or Stand options
- Winner decided based on Blackjack rules
- Easy to understand Python code

## How to Run

1. Install Python.
2. Download the project.
3. Open a terminal in the project folder.
4. Run:

```bash
python Blackjack_game.py
```

## Rules

- Try to get as close to 21 as possible.
- Going above 21 results in a bust.
- Dealer follows standard Blackjack rules.
- Highest score without busting wins.

## Technologies Used

- Python
- Random module
- Dictionaries
- Loops and Functions
